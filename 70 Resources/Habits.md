Define Habits here, they will be pulled automatically into the daily evening log. You can use the *Tasks: Create or edit tasks* command for that.

**Format**
```md
###### <habit-name>
- [ ] #habit <habit-name> | [repeat:: every day when done] [due:: today]
```
